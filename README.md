
# Plantao Juizes App - Frontend

Frontend para o APP Plantao Juizes

## Rodando localmente

Clone o projeto

```bash
  git clone https://github.com/mistermagson/job1
```

Entre no diretório do projeto

```bash
  cd job1
```

Instale as dependências

```bash
  npm install
```

Inicie o servidor

```bash
  npm run dev
```
